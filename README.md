# NeuroCord

## Это карманная нейросеть-собеседник прямо на вашем дискорд сервере!

Легко кастомизируемый и простой в использовании бот

## Инструкция по установке:

### 1. Установите .net 10.0. Скачайте исходный код
### 2. Заполните файл settings.json

Файл settings.json представляет широкий спектр различных настроек. Ниже будет дан перечень всех параметров, звёздочкой помечены обязательные к заполнению

    "configuration": {
        "connection": {
            "botToken*": Подробнее в 2.1,
            "apiKey*": Подробнее в 2.2
        },
        "messages": {
            "ping": Сообщение, выводимое командой /ping
            "resetContext": Сообщение, выводимое командой /reset-context
            "apiError": Сообщение при ошибке OpenRouter api
            "hello": Сообщение при включении бота
            "typing": Сообщение, которое бот отправляет, пока нейросеть "думает" над ответом
        },
        "neuro": {
            "model*": модель, подробнее в 2.2,
            "prompt*": Системный промпт нейросети, передайте сюда инструкции, описание характера
        },
        "settings": {
            "defaultChannelId*": Канал, в котором бот всегда читает сообщения. Пкм по каналу -> копировать ID,
            "fullServerMode": true если бот активен на всём сервере, false если только в своём канале,
            "ignoreOtherBots": true если бот отвечает только пользователям, false если и другим ботам тоже,
            "answerAllMessages": false если бот отвечает только на сообщения, где его упомянули, и на сообщения, являющиеся ответом боту, true если отвечает на все сообщения в просматриваемых каналах,
            "ignoreUserIds": передайте через запятую ID пользователей, которых бот будет игнорировать (Профиль пользователя -> ещё... -> Копировать ID),
            "ignoreMessages": передайте через запятую сообщения, на которые бот не будет отвечать (можно использовать чтобы организовать диалог двух ботов)
        }
    }
### 2.1. Создание discord приложения

перейдите на https://discord.com/developers/applications и войдите в аккаунт

Нажмите "Новое приложение" и назовите вашего бота. Кастомизируйте аватарку, описание, и прочее, как вам удобно

Перейдите в "Бот", найдите там токен, сбросьте его и сохраните. Этот токен передайте в botToken конфигурации

Перейдите в OAuth2 и поставьте в областях доступа "bot" и "applications.commands"

В правах доступа можете просто выдать администратор. Если боитесь за безопасность - вот минимальный список прав:
Отправлять сообщения
Читать историю сообщений

Ниже появится сгенерированный url. Перейдите по нему, выберите сервер, добавьте бота

### 2.2 Создание аккаунта OpenRouter

Перейдите на openrouter.ai и создайте аккаунт

Перейдите в models, выберите подходящую вам модель. Учтите, что большинство (и самые хорошие) не бесплатны, при необходимости пополните аккаунт (через сторонние сервисы из России). Скопируйте название модели и вставьте в model конфигурации

Перейдите в профиль -> API Keys

Создайте новый ключ, сохраните его и вставьте в apiKey конфигурации

### 3. Запустите проект
dotnet run

Перейдите в папку проекта в терминале, напишите команду dotnet run. Далее вы увидите логи, а затем бот напишет на сервер о готовности, если всё заполнено правильно
